# web3d-legacy-archive
Repositorio dedicado a la preservación y documentación de tecnologías web legacy como Adobe Director (DCR), Shockwave, Flash (SWF) y motores 3D asociados. Su objetivo es el análisis técnico, estudio histórico y conservación de contenidos digitales hoy inaccesibles en navegadores modernos, con fines educativos y de investigación.
